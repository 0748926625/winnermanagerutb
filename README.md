# Winner Express - Tableau de Bord Financier

Une application web moderne pour la gestion financière inspirée de votre design.

## 🎨 Caractéristiques

- **Design Moderne**: Interface élégante avec effets de verre et animations fluides
- **Palette de Couleurs Inspirée**: 
  - Vert foncé pour l'en-tête principal
  - Bleu clair pour les sections Recettes et Bilan
  - Orange clair pour les sections Dépenses et Marges
  - Vert clair pour les bénéfices positifs

## 🚀 Fonctionnalités

### 📊 Sections Principales

1. **RECETTES**: Saisissez vos revenus (Ventes, Services, Autres revenus, Subventions)
2. **DÉPENSES**: Enregistrez vos dépenses (Salaires, Loyer, Fournitures, Autres dépenses)
3. **MARGES (75%)**: Calcul automatique des marges brutes et nettes
4. **BILAN**: Vue d'ensemble avec recettes totales, dépenses totales et bénéfice
5. **OBSERVATIONS**: Section pour ajouter des commentaires et notes

### ⚡ Fonctionnalités Interactives

- **Calculs en Temps Réel**: Les totaux se mettent à jour automatiquement
- **Sauvegarde Automatique**: Les données sont sauvegardées toutes les 30 secondes
- **Génération de Rapports**: Exportez un rapport financier détaillé
- **Formatage Monétaire**: Affichage automatique en FCFA
- **Indicateurs Visuels**: Les bénéfices négatifs s'affichent en rouge

## 📱 Utilisation

1. Ouvrez `index.html` dans votre navigateur web
2. Saisissez vos données dans les champs appropriés
3. Les calculs se mettent à jour automatiquement
4. Utilisez le bouton "Sauvegarder" pour enregistrer manuellement
5. Cliquez sur "Générer Rapport" pour exporter un fichier texte

## 🛠️ Technologies Utilisées

- **HTML5**: Structure sémantique moderne

- **JavaScript ES6+**: Logique interactive et calculs
