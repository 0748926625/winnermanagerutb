# Guide d'Hébergement - Winner Express

## 🌐 Options d'Hébergement Gratuit

### 1. **GitHub Pages** (Recommandé)
- **Gratuit** et facile à utiliser
- **URL personnalisée** : `votrenom.github.io/winner-express`
- **Intégration Git** pour mises à jour faciles

#### Étapes :
1. Créez un compte GitHub : https://github.com
2. Créez un nouveau repository `winner-express`
3. Uploadez vos fichiers (index.html, script.js)
4. Allez dans Settings → Pages
5. Choisissez "Deploy from a branch"
6. Votre site sera accessible en quelques minutes!

---

### 2. **Netlify** (Très populaire)
- **Gratuit** avec SSL automatique
- **URL personnalisée** : `votresite.netlify.app`
- **Déploiement automatique**

#### Étapes :
1. Allez sur https://netlify.com
2. Créez un compte gratuit
3. Glissez-déposez votre dossier du site
4. Votre site est en ligne instantanément!

---

### 3. **Vercel** (Moderne)
- **Gratuit** et très rapide
- **URL personnalisée** : `votresite.vercel.app`
- **Excellent pour les sites statiques**

#### Étapes :
1. Allez sur https://vercel.com
2. Importez votre projet GitHub
3. Déploiement automatique à chaque modification

---

### 4. **Firebase Hosting** (Google)
- **Gratuit** avec CDN mondial
- **URL personnalisée** : `votresite.web.app`
- **Intégration Google**

#### Étapes :
1. Allez sur https://console.firebase.google.com
2. Créez un nouveau projet
3. Utilisez Firebase CLI pour déployer

---

## 🚀 Méthode la Plus Simple : Netlify

### Instructions Détaillées :

1. **Préparez vos fichiers**
   - Assurez-vous que tous vos fichiers sont dans un dossier
   - Vérifiez que le site fonctionne localement

2. **Allez sur Netlify**
   - Visitez : https://app.netlify.com/drop
   - Créez un compte gratuit

3. **Déployez**
   - Glissez votre dossier `REGIONAL 2025` dans la zone
   - Patientez 30 secondes
   - Votre site est en ligne! 🎉

4. **Personnalisez**
   - Changez le nom du site si souhaité
   - Ajoutez un domaine personnalisé plus tard
